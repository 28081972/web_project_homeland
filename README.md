# Tripleten web_project_homeland

Este proyecto tiene por finalidad dar una muestra, además del talento, de la rica historia personal y cultural de cada integrante de esta gran familia. Muchos países, muchas tradiciones y, sobre todo, muchas historias que cada persona aporta a este equipo.

Para este proyecto se utilizaron conocimientos y técnicas aprendidas a lo largo de 4 sprints anteriores, se utilizaron recursos de flexbox, enlaces de fuentes y estilos CSS por medio de la importación de archivos .CSS, se implementó la metodología BEM tanto para estructurar el código HTML, como para organizar los archivos del proyecto. Además se utilizaron recursos de diseño responsivo como media queries y valores flexibles en propiedades CSS, para garantizar la fluidez del diseño a lo largo de todo el rango de anchos de pantalla requeridos.
